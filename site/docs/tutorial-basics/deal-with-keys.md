# Know `endoflife.date` keys

There are a lot of keys used by endoflife.date; they map products to metadata such as versions, release dates and end-of-life dates.

## Products <span class="info-circle" title="A product is an item managed by geol (e.g. a package).">i</span>
```shell
geol list products
```

## Categories <span class="info-circle" title="A category groups related products (e.g. libraries, runtimes).">i</span>

```shell
geol list categories
```

## Tags <span class="info-circle" title="A tag is a short keyword used to label and filter products.">i</span>

```shell
geol list tags
```
