# Learn the `check` command
About the `check` command that helps your stack.

`geol check` analyzes a YAML-described stack and reports each component's EOL status.
Use `geol check init` to create a starter `.geol.yaml` file.
Run `geol check` to see a table of statuses and warnings.

## Create your check `yaml` file

```shell
geol check init
```
