# Fonctionnalités de `Markdown`

La documentation de `geol` prend en charge les **fonctionnalités Markdown standard** : elle est optimisée pour la variante GitHub-flavored.

## Exporter une liste de produits vers un fichier `markdown`
Par exemple :
```shell
geol list products > geol-report.md
```

## La chaîne d'outils `pandoc`

`pandoc` est un convertisseur de documents pour transformer Markdown en EPUB, PDF, DOCX, HTML, et plus encore. Flux de travail typique :

- Produire `geol-report.md` avec `geol`.
- Option A — export direct avec `pandoc` :

```bash
pandoc geol-report.md -o geol-report.epub
pandoc geol-report.md -o geol-report.pdf --pdf-engine=xelatex
```

- Option B — convertir en AsciiDoc puis utiliser Asciidoctor :

```bash
pandoc geol-report.md -f markdown -t asciidoc -o geol-report.adoc
```

Installer `pandoc` (exemple) :

```bash
brew install pandoc
```

Voir https://pandoc.org/installing.html pour les détails.
